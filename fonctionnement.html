Fonctionnement du système Domomix

Le système Domomix repose sur un fonctionnement en plusieurs étapes :

  1. Acquisition : les capteurs EnOcean (ou leur simulation) transmettent des données sous forme de trames.
  2. Transmission : les trames sont reçues via une liaison série (physique) ou par URL HTTP (simulateur web).
  3. Décodage : chaque trame est interprétée pour extraire la valeur utile (température, CO₂, etc.).
  4. Affichage : les valeurs sont affichées en temps réel dans une interface graphique.
  5. Historique : les données sont enregistrées dans un fichier CSV (version PHP) ou dans le stockage local du navigateur.

Le système web propose un simulateur pour envoyer des valeurs manuellement, les visualiser dans un tableau, les trier, les exporter ou les réinitialiser.

La version physique utilise une interface Python PyQt4/Qwt5 installée sur Raspberry Pi et reliée directement à un module EnOcean.
