Le Projet Domomix

Domomix est un projet de domotique réalisé dans le cadre de la SAE4 du BUT GEII à l’IUT de Nantes.

Notre objectif est de concevoir une solution capable de surveiller l’environnement intérieur d’une pièce (température, CO₂, humidité, etc.) à l’aide de capteurs EnOcean, et d’en visualiser les données via une interface moderne.

Le projet comporte deux versions :

  • Une version physique avec interface PyQt4/Qwt5 connectée à une Raspberry Pi
  • Une version web développée en HTML/CSS/JS avec simulateur intégré et stockage local

Ce travail met en œuvre des compétences en développement logiciel, gestion d’interface, acquisition de données, réseau et communication série/web.
