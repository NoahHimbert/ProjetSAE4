Bienvenue sur le projet Domomix

Domomix est une solution domotique développée en Python et PHP pour superviser la température ambiante à l'aide de capteurs EnOcean.

Simulation de Température

-- °C

Température aléatoire générée entre 20°C et 30°C, mise à jour toutes les 3 secondes.
